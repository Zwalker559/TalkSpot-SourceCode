# **App Name**: TalkSpot

## Core Features:

- Real-time Chat: Enable instant messaging between users with real-time updates.
- Chat Request System: Implement a system where users send chat requests via a unique ID, and recipients can accept or deny.
- User Authentication: Allow users to sign up and log in with email/password or Google, requiring email verification for unlimited chats.
- Message Deletion: Enable users to delete sent messages globally or client-side.
- Account Management (Admin): Provide admin accounts the ability to view, edit, remove, or suspend other accounts and manage sub-managers.
- Account Information Edit: Users can update their account username/password
- Media Filter Tool: Give the users the option to turn on or off the media filter. When turned on all media in chats will be blurred, when off all media will be shown normally without being blurred.

## Style Guidelines:

- Default (dark) mode: Dark magenta to dark bluish purple to navy dark blue gradient (#301934 to #483D8B to #000080).
- Light mode: Pink to light blue/dark blue mystical gradient (#FF69B4 to #ADD8E6 to #00008B).
- Body and headline font: 'Inter', sans-serif, for a clean and modern interface.
- Main screen with 'About Us' details and login/signup buttons on the top right.
